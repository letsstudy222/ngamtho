🔑 FPT.AI API Key

Lấy API key miễn phí tại console.fpt.ai → Đăng nhập → API Key → Tạo key mới · Key được lưu trong bộ nhớ trình duyệt, không gửi đi đâu khác.

🔑 Google Cloud API Key

console.cloud.google.com → Tạo project → Bật Cloud Text-to-Speech API → Credentials → API Key

🔑 Azure Subscription Key

portal.azure.com → Tạo resource "Speech" → Keys and Endpoint → Copy KEY 1
