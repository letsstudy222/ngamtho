📥 Cách cài giọng tiếng Việt trên trình duyệt

  1. Windows 10/11: Settings → Time & Language → Speech → Add voices → tìm Tiếng Việt
  2. macOS: System Settings → Accessibility → Spoken Content → System Voice → Customize → tìm Vietnamese
  3. Android: Settings → General Management → Language → Text-to-speech → cài Google TTS → tải gói tiếng Việt
  4. Sau khi cài xong, khởi động lại Chrome và tải lại trang này
